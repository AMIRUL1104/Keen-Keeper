# Keen Keeper

A smart web application to efficiently track and manage borrowed items between friends with instant updates and structured navigation.

---

## Technologies Used

- Next.js
- React.js
- Tailwind CSS
- Context API
- LocalStorage

---

## ✨ Key Features

### 1. 🔄 Dynamic Routing

- Separate dynamic pages for each friend (`/homepage/[friend]`)

### 2. ⚡ Instant UI Updates

- Global state handled using Context API for seamless data flow

### 3. 💾 Data Persistence

- Data stored in LocalStorage to retain information after reload
